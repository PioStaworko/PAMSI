# Notatnik PAiMSI
## Struktura sprawozdania
Notatnik został podzielony na dwa pliki .ipynb. Jeden z nich zawiera analizę porównawczą dla lat 2021 vs 2023.
Drugi notatnik przedstawia drugą część zadania z implementacji i predykcji wybranymi modelami.

## Uruchomienie
Aby uruchomić projekt, zbiory danych muszą znaleźć się w ścieżce ./Dane/dane_<rok>/
Następnie należy zainstalować wymagane biblioteki.
